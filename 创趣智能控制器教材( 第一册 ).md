## 第1节 人与计算机间的翻译员 - 编程软件

世界上第一台计算机于1946年发明，计算机从诞生至今，随着科学技术的飞速发展，早已成为了我们生活中必不可少的一部分。很多时候，我们都需要和计算机“打交道”，人类相互之间的交流可以通过各种语言，比如汉语，英语和法语等，`配图`我们把这些人类用于交流的语言也称为自然语言，而要想让计算机明白我们在“说什么”， 甚至能够按照我们的想法去“做什么”，就要运用到计算机语言，计算机语言可以理解是遵循一些规则的贴近于自然语言，同时又与计算机运行存在联系的语言，将用计算机语言编辑好的脚本（也叫程序）再通过编程软件的转化，就能得到真正可以操控计算机的指令。`配图`

```
我能说汉语 我能说日语。。。 语言翻译示例图
```

编程软件可以看作是人与计算机间的翻译员，将计算机语言转化成机器指令。

计算机语言的种类也很多，有C语言，Java语言和Python语言等，说到这些语言我们想象到的可能会是一段密密麻麻的代码，一个个还不懂得英文单词。`配图`  看到这，你可能会有些畏难的心理，然而，我们在后续课程中真正要学习的是一种图形化的编程语言`配图` , 编写程序的过程，就像是拼搭积木的过程，找到对应的积木模块，放到核实的位置组合起来。编写过程是非常简单，但能实现的功能不容小视，利用它我们可以制作动画、游戏，还可以控制电子器件做一些有趣的发明创造。

## 第2节 让计算机同世界问好

在计算机屏幕上输出"hello world"（中文含义：“你好世界”）这句话的程序，于世界著名的编程书籍《The C Programming Language》作为第一个案例提出。后来，广泛流传，基本上任意一门计算机语言的学习都将其作为第一个示例程序。`图示`

```

```

同样，这节课，我们也要尝试编写第一个程序，并在屏幕上显示“你好世界”。

如何编写一段脚本（程序）呢？

在创趣编程软件中，将构建脚本需要的单个模块从模块区，拖拽到脚本区，对应模块的组合，即完成了编写的过程。当然，编写也同样需要讲究一些规则，这个就好比我们传统的代码编程中的语法，但这里的“语法”大多数都不需要我们去特别记忆，也是一种图形化的展现形式。我们在拼搭积木时，并不是所有的积木都能连接在一起，积木会存在不同的形状不同的连接位置。软件中的模块亦是如此，一个符合“语法”规则的程序，会在脚本区呈现一种金米相连的状态`图示`，并且当其中一个模块靠近另一个模块时，你会发现像是有磁力作用一样，被吸附在一起；而不符合“语法”规则的程序，是无法被衔接在一起的`图示`

```

```

选中角色“小创”，对他进行脚本编写。从模块区的外观模块库中拖拽出一条`图示`模块，放置在脚本区，将可编辑文本框中默认的“你好”，修改为“你好世界”。这样，我们的第一条程序就编写完毕了。

```

```

想要运行脚本区的一段程序，我们可以直接点击它，正在运行的程序会在其外轮廓呈现黄色的高亮显示状态。点击程序运行，你发现了什么？

舞台区小创说话了！内容便是“你好世界”，大功告成。

```

```

但是，我们好像并没有看见程序运行时黄色的外轮廓，这又是为什么呢？

原来，程序执行的速度是非常快的，这样一条较为简单的程序，真正运行完毕眨眼的功夫都不要，那我们自然就很难发现了。在后续运用到“等待”，“循环”等模块后，我们可以再来观察，你就能看见表示正在运行状态的黄色外轮廓框了。

常见的运行程序的另一种方式是，可以添加一个`图示`在程序顶端。此时，想要运行程序，我们依然可以通过直接点击程序本身的方式，除此之外，我们还可以点击舞台区左上角的“绿旗”图标`图示`，当我们点击绿旗时，所有以绿旗开头的程序都会开始运行。

```

```

要终止正在运行的程序，如果仅仅是某一段，可以直接再次点击程序本身，此时，应该看到的现象是，代表着正在运行的黄色外轮廓会消失。当然，对于我们刚刚编写的“你好世界”，是不需要主动进行终止的，因为它在开始运行的一瞬间就完成了所有的部分。`图示`与这种方法对应的，我们也可以通过点击舞台区“红色圆形”按钮来终止正在运行的程序，只不过这时，会将所有正在运行的程序都停止下来。
